* Kalray DMA NOC controller tests

Required properties:
- compatible: Should be "kalray,k1c-dma-noc-test"
- dmas: dma_noc used
- dma-names: dma name
- iommus: if defined, same iommu for dma_noc

Example:

/* DMA NOC*/
test_dmaengine@0 {
	compatible = "kalray,k1c-dma-noc-test";
	dmas = <&dma_noc 1>;
	dma-names = "k1c-dma-noc";
	iommus = <&iommu_noc>;
};
